# Как запустить?
Установить компилятор плюсов, затем:

для ООП версии:
```sh
$ g++ -std=c++17 -o game main.cpp -lncurses
```

для Функциональной версии:
```sh
$ g++ -std=c++17 -o game functional_game.cpp -lncurses
```

Запускаем скомпиленный файл

```sh
$ ./game
```
